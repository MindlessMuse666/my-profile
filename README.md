## 🎨 Особенности

- 🌈 **Динамический фон**: Хацунэ Мику шапкирована и готова к раскидке сельдерея на Дасте!

- 🐱 **Падающие няшности**: Огромный массив из 70+ файлов - няшновое воздаяние!
- 💖 **Любовь**: Весь проект пропитан теплом и уважением к одному [очень перспективному менчику](https://github.com/MindlessMuse666 "Катище")!
